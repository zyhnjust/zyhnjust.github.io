读书笔记 Good Data and Machine Learning

这篇文章告诉大家关于数据的重要性。

前言

  • 机器学习不只是学习一些课程或者读几本书， 然后知道如何创建CNN 来解决一个手写数字的辨认问题就完事了。 不可以忽略两个最重要的点： 数据和数学。

    Machine Learning is, after all, Data Driven AI, and your model will be only as good or as bad as the data you have.

特征的选择

  • 如果数据不相关，任何模型都没有用的。 所以甄别有用的数据非常重要
  • 总的来讲， 要选择相关性高的特征， 去除无关的特征项。

    图像法。

    你可以根据图像判断是否这个特征和结果是否关联。

    数学方法。

    比如几种协方差排序

    Pearson’s Correlation Coefficient helps you find out the relationship between two quantities. It gives you the measure of the strength of association between two variables. The value of Pearson’s Correlation Coefficient can be between -1 to +1.

If you try to train a model on a set of features with no or very little correlation, you will get inaccurate results.
Instead, it’s better to use fewer highly correlated features to train a model.

hzhang： 准备选择一些， 然后排除不太关联的。

其他的关联

  • Spearman rank order correlation
  • Pearson’s Rank Correlation.
    hzhang： 还有一个叫kendall
    准备把几个相关系数都计算， 然后选择

hzhang: 准备用top的一些特征来计算。

注意： 如果你的数据集比较大， 那么如果相关性较小， 比如0.4 也可能是可以训练出好的模型的。

hzhang： 我们的最高0.3 左右。

模型选择

就是说线性关系的用线性模型，用CNN不一定好使。 这个对我们还好， 我们是广选材， 就是都试试， 选择好的。 毕竟还不清楚是线性的还是别的关系。
