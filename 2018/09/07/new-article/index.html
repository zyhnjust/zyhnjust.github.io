1.1 jquery 中 $(document).ready()

window.onload $(document).ready()
必须等待网页所有内容都加载完毕 这个是DOM 绘制完成？ 可能有些相关的没有加载完成
不可以同时写多个。 如果多个只是最后一个执行 每次都执行
没有简化做法 $(function(){ xxx });

1.2 jquery 对象和DOM 对象

dom 对象是直接从网页中获取的
document。getElementById(“foo”).innerHTML;

jquery对象是包装之后的对象

#(“#foo”).html();

注意两个对象的方法不同， 不可以串用。
然后可以互转

var $variable = jquery 对象

var #cr = $(“$cr”); //jquery obj
var cr = $cr[0]; //get dom obj

从dom 到jquery

var cr = document.getElementById(“cr”);
var $cr = $(cr); // this will generate jquery obj
